# LOF Arbiter - 套利精灵 Skill

> LOF 基金溢价套利机会监测与交互式咨询

## 功能说明

本 Skill 帮助用户查询和分析 LOF 基金的折溢价套利机会。

### 核心能力

1. **套利机会查询** - 查询高溢价/折价 LOF 基金 TOP 排行
2. **单基金诊断** - 输入基金代码/名称，返回详细信息和分析
3. **套利收益测算** - 计算扣除费用后的理论套利收益
4. **操作建议** - 根据溢价率、流动性、申购状态给出买卖建议

### 数据来源

- DataHub ETL Pipeline 产出的 `dwd_fund_lof` 融合表
- 数据库路径：`/Users/jackyang/.openclaw/workspace/DataHub/datahub.db`

### 触发关键词

| 用户意图 | 典型问法 |
|----------|-----------|
| 查询机会 | 今天有什么套利机会、哪些 LOF 溢价率高 |
| 基金诊断 | 帮我看看 160140、XX 基金怎么样 |
| 收益测算 | 套利收益怎么算、买入 10 万能赚多少 |
| 操作建议 | 能买吗、能卖吗、现在适合操作吗 |

## 使用方法

### 1. 查询今日套利机会

```
用户：今天有什么套利机会？
```

Skill 会返回：
- 高溢价 TOP10（卖出赎回套利机会）
- 高折价 TOP10（买入套利机会）
- 每个品种标注：溢价率、现价、净值、成交额
- 风险提示

### 2. 查看特定基金

```
用户：帮我看看 160140
用户：招商全球优选怎么样
```

Skill 会返回：
- 基金基本信息
- 溢价率计算
- 流动性分析
- 操作建议

### 3. 套利收益测算

```
用户：买入 10 万的 160140，持有 7 天后卖出，能赚多少？
```

Skill 会计算：
- 买入费用（申购费）
- 持有期间净值变化（如有）
- 卖出费用（赎回费 + 佣金）
- 理论净收益

### 数据库表结构

参考 `dwd_fund_lof` 表：

| 字段 | 说明 |
|------|------|
| 基金代码_full | 标准代码：XXXXXX.SZ / XXXXXX.SH |
| 基金名称 | 基金简称 |
| 现价 | 当前交易价格 |
| 净值 | 最新单位净值 |
| 溢价率 | 自动计算：(现价-净值)/净值×100% |
| 成交额 | 今日成交金额 |
| 涨跌幅 | 今日涨跌幅 |
| 申购状态 | 开放式/暂停申购等 |
| 手续费 | 申购费率 |
| 交易日期 | 数据日期 |

### 关键计算

**溢价率**：
```
溢价率(%) = (现价 - 净值) / 净值 × 100
```

**套利收益估算**：
```
买入费用 = 买入金额 × 申购费率
赎回费 = 买入金额 × 赎回费率（如持有<7天则1.5%，否则0.5%）
佣金 = 卖出金额 × 佣金费率
净收益 ≈ 卖出金额 - 买入费用 - 赎回费 - 佣金
```

## 状态说明

| 状态 | 含义 | 操作建议 |
|------|------|----------|
| 高溢价 (>1%) | 卖出赎回有利 | 可考虑卖出 |
| 低折价 (<-1%) | 买入套利有利 | 可考虑买入 |
| 正常 (-1%~1%) | 无明显机会 | 观望 |
| 暂停申购 | 无法套利 | 关注开放时间 |

---

## 注意事项

1. **T+2 交割**：LOF 套利从买入到资金到账需 2 个交易日
2. **净值估算**：收盘后净值 T+1 更新，溢价率为估算值
3. **流动性风险**：高溢价不等于能成交，需关注成交额
4. **费用结构**：不同基金费率不同，以实际为准
