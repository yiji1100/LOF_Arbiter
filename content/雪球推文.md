# 雪球推文：LOF套利精灵

## 标题
【开源】用Python做了一个LOF套利工具，自动筛选限购溢价机会

## 正文

最近开源了一个LOF套利分析工具：**LOF Arbiter**

GitHub：https://github.com/yiji1100/LOF_Arbiter

---

## 解决的问题

1. 手动刷行情软件效率低
2. 溢价率计算容易出错
3. 不知道哪些基金限购/暂停申购
4. 净值日期容易混淆

---

## 核心功能

**1. 限购溢价优先筛选**

核心逻辑：限购+高溢价=优质套利机会

原因：
- 限购产品溢价更稳定
- 无限购的流动性太好，溢价稍纵即逝

**2. 净值日期标注**

每条数据都标注净值日期，避免用错数据做决策

**3. 多维度排序**

- 限购高溢价 TOP
- 高溢价 TOP（卖出赎回）
- 高折价 TOP（买入机会）
- 全量排行

## 使用方式

```bash
# 克隆代码
git clone https://github.com/yiji1100/LOF_Arbiter.git
cd LOF_Arbiter

# 安装依赖
pip install akshare pandas

# 获取当日数据
python scripts/etl.py

# 查询套利机会
python scripts/query.py
```
