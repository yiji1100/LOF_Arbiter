# 雪球推文：LOF套利精灵

## 标题
【OpenClaw Skill】做了一个LOF基金行情监控工具，支持限购基金筛选

## 正文

最近开源了一个基于OpenClaw Skill的LOF基金行情监控工具：**LOF Arbiter**

GitHub：https://github.com/yiji1100/LOF_Arbiter

---

## 什么是OpenClaw Skill？

OpenClaw是一个AI助手框架，Skill是它的能力扩展模块。装上这个Skill，AI就能帮你监控LOF基金行情。

---

## 解决的问题

1. 手动刷行情效率低
2. 限购基金容易被忽略
3. 净值日期容易混淆
4. 数据分散不好整理

---

## 核心功能

**1. 限购基金优先展示**

为什么重要？
- 限购基金每天买入金额有限制
- 大资金无法快速进出，价格更稳定
- 更适合作为投资参考

**2. 对话式交互**

像聊天一样问AI：
- "今天LOF有什么机会？"
- "帮我看看黄金LOF"
- "导出今天的行情"

**3. CSV导出**

一键导出完整数据，方便深度分析

## 使用方式

```bash
# 克隆代码
git clone https://github.com/yiji1100/LOF_Arbiter.git
cd LOF_Arbiter

# 安装依赖
pip install akshare pandas

# 安装OpenClaw Skill
openclaw skills install ./lof-arbiter

# 开始对话！
```
